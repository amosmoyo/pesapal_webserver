# Web server

> Problem 1: Web server (with optional secure communication).

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run start:dev

# Run in prod mode
npm start
```

## TLS (HTTPS) or a secure protocol
> Used openssl (to use go to chrome://flags/ and enable invalid certificates)
